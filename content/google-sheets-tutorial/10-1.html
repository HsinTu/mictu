本章示範檔案：Chapter 10. 函式 (五)：特殊功能函式
本節將會介紹一些功能較為特殊的函式，前五節包括會計、資料生成、資料型態與格式相關的函式，最後一節會介紹資料庫函式，提供一種不同的方式完成第 6.5~6.6 節學到的資料統計值，大家可以根據自己的習慣選擇使用的函式。另外因為各節的函式用途不太相同，因此本章沒有整合各節內容的進階應用。

Google Sheets 提供四種計算資產折舊費用、一種計算累計折舊的函式，分別如下：

  • SLN / DDB / DB / SYD：使用直線法 / 倍數餘額遞減法 / 定率遞減法 / 年數合計法計算資產折舊費用

語法

  • SLN (費用, 殘值, 期限)
  • DDB (費用, 殘值, 期限, 周期, [係數])
  • DB (費用, 殘值, 期限, 周期, [月])
  • SYD (費用, 殘值, 期限, 周期)

輸入

  • 費用：資產的原始成本。
  • 殘值：折舊結束時的資產價值。
  • 期限：資產折舊的總期數。
  • 周期：要計算第幾期的。
  • 係數：倍數餘額遞減法的倍數，預設為 2。
  • 月：定率遞減法折舊第一年的月數，預設為 12。

輸出

根據給定的輸入，計算直線法 / 倍數餘額遞減法 / 定率遞減法 / 年數合計法計算資產折舊費用。

  • VDB：計算倍數餘額遞減法的累計折舊

語法

VDB (費用, 殘值, 期限, 開始期間, 結束期間, [係數], [不切換])

輸入

  • 費用 / 殘值 / 期限 / 係數：與 DDB 相同。
  • 開始期間 / 結束期間：計算第幾期末到第幾期末的累計折舊，例如想計算前三年的累計折舊，且一期為 1 年，就使用 0 / 3。
  • 不切換：TRUE / FALSE，預設為 FALSE，當折舊大於遞減餘額計算時，是否切換到直線折舊法。

輸出

使用倍數餘額遞減法計算指定期間內的資產折舊值。
