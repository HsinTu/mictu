2.1 基本欄列操作

  • 調整欄 / 列的大小：
    • 拖曳調整大小：將鼠標放在數字或欄位換欄 / 列處，鼠標會變顏色，如下圖：
    • 自動調整成內容大小：與 Google Sheets 相同。
    • 設定確切欄 / 列大小：「選擇要調整的欄 / 列」 ⭢ 按滑鼠右鍵點選「調整欄 / 列的大小」 ⭢「輸入想要的寬度 / 高度」 ⭢「確定」
  • 凍結：點選上方工具列的「檢視」後，會出現「凍結窗格」、「凍結頂端列」、「凍結首欄」，其中凍結窗格會根據目前所在的儲存格同時凍結欄 / 列
  • 插入 / 刪除：與 Google Sheets 相同，此外也可以在「常用」的右側找到「插入」及「刪除」，「格式」中則有設定欄 / 列大小、隱藏欄位等功能。
  • 移動：小手的符號需要在靠近欄 / 列邊界的地方會出現，且移動到的儲存格原本有內容時，會直接取代原本的資料。
  • 隱藏：與 Google Sheets 相同，亦可在「常用」的「格式」中進行設定。
  • 將欄 / 列分組：點選上方工具列的「資料」後，會出現「群組」、「取消群組」，只要選取要分組 / 取消分組的欄 / 列後，點選「群組 / 取消群組」即可。

2.2 定義已命名範圍

在 Excel 中已命名範圍較常稱呼為「名稱」，且功能較為完善。定義方式有以下兩種：

  • 使用名稱方塊快速命名：與 Google Sheets 相同，點選上下箭頭可查看目前已命名的範圍，點選後會快速跳轉至該名稱的範圍，但不能調整與編輯。
  • 使用「定義名稱」：和 Google Sheets 的「已命名範圍」視窗功能類似。點選上方工具列的「公式」後，會出現以下與「名稱管理員」相關的項目，各功能說明如下：
    • 定義名稱：點選後可以建立新的名稱，其中「範圍」可以選取該名稱要是用在哪幾張工作表，若每一張工作表都能使用該名稱則選擇「活頁簿」即可，如下圖：
    • 在公式中使用：點選後可快速建立相關的公式，不過一般來說更常直接在輸入公式時使用，因此可以忽略。
    • 從選取範圍建立：可一次命名多欄 / 列的範圍，步驟為「選取要命名的範圍 ⭢ 從選取範圍建立 ⭢ 選取命名的依據 (如下圖) 並點選確定」即可，如下圖：

  • 「名稱管理員」視窗：等同於 Google Sheets 的「已命名範圍」視窗，但功能更深入，點選後如下圖：

    「名稱管理員」視窗大致有以下功能：
    • 新增名稱：點選左上角的新增即可。
    • 編輯、刪除名稱：只要點選要編輯 / 刪除的範圍即可進入設定，若只有要更改範圍可點選下方「參照」右邊的 icon 即可。
    • 排序名稱：可以依照名稱、值、參照別、範圍、註解排序，只要點選圖中的表頭即可遞增或遞減排序 (按一下切換遞增 / 遞減)。
    • 篩選現有的條件：點選右上角的「篩選」即可針對現有的條件進行篩選。

2.6 資料驗證

Excel 的資料驗證在工具列的「資料」中，點選「資料驗證」便可進入資料驗證視窗。

  • 設定：設定資料驗證的規則，可發現 Excel 的規則相對較少，但亦可使用「自訂」輸入公式來滿足，例如「文字包含 Google」可使用「=NOT(ISERROR(SEARCH(("Google"),(C5))))」代替，而檔案從 Google Sheets 下載時便會自動進行轉換。

2.7 快捷鍵

除了「僅貼上值」之外，第 2.7 節介紹的其他快捷鍵皆適用，而除此之外 Excel 支援更多快捷鍵功能，詳見官方文件的說明。其中 Windows 的快捷鍵比 Mac 多出許多。此外，Windows 的 Excel 更支援自訂快捷鍵的功能，這部分亦詳見官方文件的說明。

綜合以上，Excel 在儲存格操作與格式設定方面，功能較 Google Sheets 齊全很多，尤其是「表格」的型態與自訂快捷鍵的部分，是目前 Excel 得以高效率運作的一大關鍵！
